### java.lang.String

- `indexOf()`

  Returns the index of the first occurrence of the specified character or substring.

```java
String str = "Hello, You!";
int index = str.indexOf("You");
// Outputs: 7
```

1. **toCharArray()** :

   _Converts the string into a character array._

```java
String str = "Pranshu";
char[] chars = str.toCharArray();
// Converts "Pranshu" to ['P', 'r', 'a', 'n', 's', 'h', 'u']
```

`charAt()`

Returns the character at the specified index.

```java
String str = "Pranshu";
char c = str.charAt(1);
// Outputs: 'r'
```
